# КМ Сканер — Честный знак

Браузерное PWA-приложение для сканирования кодов маркировки через камеру телефона и экспорта в CSV для портала Честного знака.

## Использование

1. Открой приложение на телефоне (HTTPS обязателен для камеры)
2. Нажми **▶ Старт** — разреши доступ к камере
3. Наводи на DataMatrix-коды на этикетках
4. Коды автоматически добавляются в список
5. Нажми **⬇️ Скачать CSV** и загрузи файл на портал ЧЗ

## Запуск

### GitHub Pages (рекомендуется — HTTPS бесплатно)
Залей содержимое этой папки в GitHub репозиторий и включи GitHub Pages.

### Локально (только для проверки на компьютере)
```
python -m http.server 8080
# Открой: http://localhost:8080
```

> ⚠️ Камера работает только через HTTPS или на localhost.
> На телефоне через локальную сеть нужен HTTPS.
